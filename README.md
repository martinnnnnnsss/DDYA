## Semana 7
- Pila y Colas: Son estructuras de datos lineales que organizan elementos según el orden en que se insertan y eliminan. Las pilas funcionan con el principio LIFO (Last In, First Out), mientras que las colas funcionan con FIFO (First In, First Out).
- Listas enlazadas: Son estructuras de datos donde cada elemento contiene un valor y una referencia al siguiente nodo, formando una cadena. Permiten insertar y eliminar elementos fácilmente sin reorganizar toda la estructura.
